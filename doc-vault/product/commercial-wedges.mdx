## Best initial wedge

Start with interactive AI reports for agents.

Then expand:

```text
Interactive AI reports
-> live agent dashboards
-> canvas as API
-> agentic applets
-> open protocol / self-hosted ecosystem
```

## Hosted Canvas Hub

For developers and teams.

Includes canvas storage, share links, permissions, version history, live updates, API exports, and embeds.

Pitch:

> Hosted infrastructure for agent canvases.

## Agent Report Kit

For dev teams and AI builders.

Includes schema, renderer, SDK, CLI, templates, and examples.

Pitch:

> Make your agent outputs look like real product surfaces.

## Branded AI Report Links

For agencies, consultants, and marketers.

Includes custom branding, report templates, share links, client-facing viewer, analytics, and future PDF/export support.

Pitch:

> Deliver AI-generated audits as interactive client reports.

## Canvas Applets

For advanced users building live agentic workflows.

Includes live widgets, actions, forms, agent callbacks, and workflow state.

Pitch:

> Build small agentic apps without building a full app.

## Self-hosted Knify Hub

For technical and privacy-sensitive teams.

Includes own Hub, own links/domains, own storage, custom auth, and private widgets.

Pitch:

> Use the Knify protocol inside your own infrastructure.
